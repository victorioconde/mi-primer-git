Comandos para git

  1. Primer comando: git init (Para inicializar git)
  2. Segundo comando: git status (Para chequear el estado de git en nuestro proyecto)
  3. Tercer comando: git add . (Para trackuear todos los archivos)
  4. Cuarto comando: git commit -m "Mensajito" (Para hacer el paquetito de una version, que podemos subir a la nube)
  5. git branch -M main
  6. git remote ....
  7. git push
